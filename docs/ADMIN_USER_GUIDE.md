# Инструкция по админ-панели H-Sport

Краткое руководство для сотрудников: как пользоваться админкой.

---

## Вход

- Адрес: **ваш-сайт.ru/admin**
- Вход под учётной записью с ролью **Администратор**. Без прав админа панель недоступна.

В левом меню: Главная, Заказы, Товары, Промокоды, Рассылки, Шаблоны писем, Пользователи, Метрики.

---

## Главная (Дашборд)

- Сводка по выручке, заказам, метрикам за выбранный период.
- Переключатель периода (день/неделя/месяц и т.п.).
- Графики и карточки: можно оценить текущую активность магазина.
- Быстрые ссылки на разделы (заказы, товары и т.д.).

---

## Заказы

- **Список заказов** — таблица с номером, датой, статусом, суммой, способом оплаты/доставки, контактом.
- **Фильтры:** по статусу, по email. Кнопка «Применить» / «Сбросить».
- **Экспорт** — выгрузка заказов (если есть кнопка экспорта).
- **Переход в заказ** — клик по строке или кнопка «Открыть» открывает карточку заказа: состав, оплаты, адрес доставки. Статус здесь только просматривается (изменение статуса — если есть отдельная кнопка/действие в интерфейсе).

Используйте раздел, чтобы искать заказы, проверять оплаты и данные доставки.

---

## Товары

- **Список товаров** — название, артикул (SKU), категория, цена, наличие и т.п.
- **Фильтры** — по категории, поиск по названию/артикулу. Пагинация внизу.
- **Добавить товар** — кнопка «Новый товар» ведёт на форму создания: название, описание, цены, варианты (размеры/цвета), фото. Сохранить — товар появится в каталоге.
- **Редактировать** — кнопка с иконкой карандаша у строки: переход в редактирование товара (те же поля). Можно раскрыть строку и править варианты в таблице.
- **Удалить** — кнопка с иконкой корзины. Подтверждение перед удалением.
- **Копировать SKU** — при необходимости скопировать артикул в буфер обмена.

Удобно вести каталог, добавлять новинки и снимать с продажи через удаление или отключение.

---

## Промокоды

- **Список** — код, скидка/тип скидки, период действия, активен/неактивен.
- **Фильтры:** по коду, по состоянию (все / только активные / только неактивные).
- **Создать промокод** — кнопка открывает форму: код, тип скидки (процент или фикс), сумма/процент, даты начала и окончания, лимит использований (если есть). Сохранить — промокод можно выдавать клиентам.
- **Редактировать** — изменить условия или срок действия.
- **Удалить** — удаление промокода.

В списке можно смотреть активные и неактивные промокоды; просроченные по дате не применяются при оформлении заказа.

---

## Рассылки

Два подраздела (вкладки): **Подписчики** и **Выпуски**.

- **Подписчики** — таблица email-адресов подписчиков, источник подписки (подвал сайта, попап, оформление заказа), подтверждён ли email. Фильтры: по email, по статусу подтверждения. При необходимости — удалить подписчика (отписать).
- **Выпуски** — список выпусков рассылки. **Создать выпуск** — указать тему и содержимое письма. **Отправить** — отправка выпуска подписчикам. Отправка делается по кнопке для выбранного выпуска.

Используйте для email-маркетинга и управления подпиской.

---

## Шаблоны писем

- Просмотр превью системных писем (подтверждение email, сброс пароля и т.д.). Выбор шаблона по вкладкам/кнопкам. Только просмотр; изменение шаблонов — через разработчика/конфигурацию.

---

## Пользователи

- **Список пользователей** — email, имя, роль, дата регистрации.
- **Поиск** — по email или имени. Пагинация.
- **Переход в карточку пользователя** — просмотр данных и при необходимости смена роли (если есть кнопка «Изменить роль» / аналогичное действие). Обычно здесь выдают права администратора или снимают их.

Не раздавайте роль администратора без необходимости.

---

## Метрики

- **Общие метрики** — загрузка API, базы, производительность (если включено).
- **Web Vitals** — показатели скорости страниц (LCP, FID, CLS и т.п.).
- **Дашборд метрик** — детальные графики и отчёты по производительности и бизнес-показателям.

Нужны для оценки работы сайта и API; детали зависят от настроек на сервере.

---

## Поддержка

Внизу бокового меню — ссылка «Техническая поддержка» (почта). По техническим сбоям, доступу и настройкам обращаться туда или к ответственному за сайт.
